## ğŸ”§ Troubleshooting

### Common Issues

**Q: Animations are laggy on older devices**
A: Use `.drawingGroup()` modifier on complex views, reduce particle counts for confetti on iPhone 8 and below.

**Q: Haptic feedback not working**
A: Check device supports haptics (iPhone 7+). Test with `UIImpactFeedbackGenerator(style: .medium).impactOccurred()`.

**Q: Push notifications not delivering**
A: Verify certificate in Apple Developer portal, check user granted permission, test on physical device (not simulator).

**Q: StoreKit purchases failing in TestFlight**
A: Ensure sandbox accounts are set up, clear iTunes cache, use real Apple ID (not test email).

**Q: CloudKit sync not working**
A: Enable CloudKit in Xcode capabilities, verify container ID matches, check user is signed into iCloud.

### Performance Optimization

- **Lazy loading**: Use `LazyVStack`/`LazyVGrid` for long lists
- **Image caching**: Cache emoji/icons to avoid redraws
- **Animation throttling**: Limit confetti particles on low-end devices
- **Database indexing**: Add Core Data indexes on frequently queried fields
- **Memory profiling**: Use Instruments to catch leaks
